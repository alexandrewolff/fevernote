# Fevernote

Fevernote is a note taking web app, with authentication.

It is made of a backend node server, and a frontend react application.

## Run it

1. Run the backend

```
cd backend
```

```
npm run start
```

2. Build the frontend

```
cd ../frontend
```

```
npm run build
```

3. Open the built web app

```
open build/index.html
```
